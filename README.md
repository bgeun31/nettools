# 네트워크 도구 모음 웹

## 관련 명령어

- python 모듈 일괄 설치(최초 1회만 실행)
```sh
pip install -r requirments.txt
```

- React 라이브러리 일괄 설치(최초 1회만 실행)
```sh
npm install
```

- 백엔드 서버(FastAPI) 실행
```sh
uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload
```

- 프론트 서버(웹) 실행
```sh
npm run dev
```

- localhost:3000 접속
```sh
http://localhost:3000/
```
